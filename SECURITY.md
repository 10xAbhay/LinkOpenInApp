# Security Policy

## Supported Versions

Only the latest version of LinkOpenInApp receives security updates.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security issue, please do not disclose it publicly.

Instead, contact the project maintainer with:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

We will acknowledge reports promptly and work toward a timely resolution.
