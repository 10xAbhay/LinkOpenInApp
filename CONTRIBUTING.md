# Contributing

Thank you for your interest in contributing to LinkOpenInApp.

## Getting Started

1. Fork the repository.
2. Create a new branch.

```bash
git checkout -b feature/your-feature
```

3. Make your changes.
4. Test your changes.
5. Commit using a clear message.

```bash
git commit -m "feat: add support for YouTube Shorts"
```

6. Push your branch.

```bash
git push origin feature/your-feature
```

7. Open a Pull Request.

## Pull Request Guidelines

- Keep pull requests focused on a single feature or fix.
- Follow the existing code style.
- Write clear commit messages.
- Update documentation when necessary.

## Reporting Bugs

Please include:

- Browser
- Device
- Steps to reproduce
- Expected behavior
- Actual behavior

## Feature Requests

Feature requests are welcome. Please create an issue describing the problem and proposed solution before implementation.
